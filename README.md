# Sistema de Facturación Electrónica

Proyecto desarrollado con Node.js, Express, PostgreSQL y Prisma ORM.

## Descripción

Sistema de facturación electrónica orientado a pequeñas y medianas empresas. El proyecto permite la gestión de clientes, productos, facturas y usuarios mediante una API REST.

## Tecnologías

* Node.js
* Express.js
* PostgreSQL
* Prisma ORM
* Thunder Client
* Git y GitHub

## Funcionalidades implementadas

* Conexión a PostgreSQL
* Integración con Prisma ORM
* API REST
* Gestión de clientes
* Creación y consulta de clientes

## Estructura del proyecto

src/
├── config/
├── controllers/
├── middlewares/
├── routes/
├── services/
└── utils/

## Instalación

npm install

## Ejecución
